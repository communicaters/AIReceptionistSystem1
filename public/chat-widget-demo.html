AI Receptionist Chat Widget

A lightweight, customizable chat widget that can be embedded on any website

Integration Instructions

To add the AI Receptionist chat widget to your website, simply add the following script tag to your HTML before the closing </body> tag:

<!-- AI Receptionist Chat Widget -->
<script>
  // Configuration (Optional)
  window.AR_SERVER_URL = "https://your-server-url.com"; // Your AI Receptionist server URL
  window.AR_WIDGET_TITLE = "Your Company Name";         // Chat widget title
  window.AR_PRIMARY_COLOR = "#2563eb";                  // Primary color (hex code)
  window.AR_GREETING_MESSAGE = "Hello! How can I help you today?"; // Initial greeting
</script>
<script src="https://your-server-url.com/embed-chat.js"></script>

Configuration Options

The following configuration options are available:

window.AR_SERVER_URL

The URL of your AI Receptionist server. If not specified, the script will use the current origin.

window.AR_WIDGET_TITLE

The title displayed in the chat header. Default: "AI Receptionist"

window.AR_PRIMARY_COLOR

The primary color used for the chat bubble, header, and buttons. Default: "#2563eb" (blue)

window.AR_GREETING_MESSAGE

The initial greeting message displayed when the chat is opened. Default: "Hello! How can I assist you today?"

Security & Privacy

The chat widget includes the following security features:

  • Unique session IDs for each visitor, stored in localStorage
  • Secure WebSocket connection (wss:// on HTTPS sites)
  • No sensitive data is exposed in the console or DOM
  • Automatic reconnection with exponential backoff
  • Heartbeat mechanism to keep connections alive
